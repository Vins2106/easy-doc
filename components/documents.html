Documentation


getDocById()

this help you to get specify element by id.


Usage

query: element id

getDocById(elementid)

return: HTML Object

getQS()

this help you to get specify element with query selector


Usage

query: id, class, tag, etc

getQS(query)

return: HTML Object

getQSAll()

this help you to get specify elements with query selector


Usage

query: id, class, tag, etc

getQSAll(query)

return: HTML Node Object

getDocsByClass()

this help you to get specify elements with class name


Usage

query: class name

getDocsByClass(className)

return: HTML Node Object

getDocsByName()

this help you to get specify elements with name


Usage

query: name

getDocsByName(name)

return: HTML Node Object

getDocsByTagName(()

this help you to get specify element with tag name


Usage

query: tag name

getDocsByTagName(tagName)

return: HTML Node Object

getDocsByTagNameNS()

this help you to get specify element with query selector


Usage

query: name space uri && tag name

getDocsByTagNameNS(nameSpaceURI, tagName)

return: HTML Node Object

Example


<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>My Website</title>
</head>
<body>
	<!-- id to add innerHTML -->
	<div id="demo"></div>
	
	<!-- our cdn -->				
	<script src="https://vins2106.github.io/easy-doc/libs/cdn/v1/index.js"></script>
	<script>
	getDocById("demo").innerHTML = "My website"
	</script>
</body>
</html>
