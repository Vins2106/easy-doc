Documentation


getDocById()

this help you to get specify element by id.


Usage

query: element id

getDocById("my-id")

return: HTML Object

getQS()

this help you to get specify element with query selector


Usage

query: id, class, tag, etc

getQS("my-class")

return: HTML Object

getQSAll()

this help you to get specify elements with query selector


Usage

query: id, class, tag, etc

getQSAll("my-class")

return: HTML Node Object

getDocsByClass()

this help you to get specify elements with class name


Usage

query: class name

getDocsByClass("class-name")

return: HTML Node Object

getDocsByName()

this help you to get specify elements with name


Usage

query: name

getDocsByName("a-name")

return: HTML Node Object

getDocsByTagName()

this help you to get specify element with tag name


Usage

query: tag name

getDocsByTagName("h1")

return: HTML Node Object

getDocsByTagNameNS()

this help you to get specify element with query selector


Usage

query: name space uri && tag name

getDocsByTagNameNS(nameSpaceURI, "h2")

return: HTML Node Object

addClassTo()

this help you to add multiple class to a element


Usage

query: HTML Object && []classes

addClassTo(element, ["class", "name"])

return: None

removeClassFrom()

this help you to remove multiple class to a element


Usage

query: HTML Object && []classes

removeClassFrom(element, ["class", "name"])

return: None

getAttrFrom()

this help you to get a attribute value from a element


Usage

query: HTML Object && Attribute Name

getAttrFrom(element, "title")

return: Attribute Value

getAttr()

this help you to get specify element attributes


Usage

query: HTML Object

getAttr(element)

return: Attribute List

setAttrTo()

this help you to set a attribute to a element


Usage

query: HTML Object, Attribute name, Attribute value

setAttrTo(element, "title", "this is title")

return: Attribute Value

show()

this will show a element


Usage

query: HTML Object, Display Type

show(element, "flex")

return: Display Type

showAfter()

this will show a element after seconds


Usage

query: HTML Object, Seconds, Display Type

showAfter(element, 5, "flex")

return: String && Display Type

hide()

this hide a element


Usage

query: HTML Object

hide(element)

return: Display Type

hideAfter()

this hide a element after seconds


Usage

query: HTML Object, Seconds

hideAfter(element, 5)

return: Display Type

Utility functions


random()

this will generate random number between [min] - [max]


Usage

query: MIN, MAX

random(1, 10)

return: random number between [min] - [max]

scrollTop()

this will scroll the web page to top


Usage

query: None

scrollTop()

return: None

scrollBottom()

this will scroll the web page to bottom


Usage

query: None

scrollBottom()

return: None
